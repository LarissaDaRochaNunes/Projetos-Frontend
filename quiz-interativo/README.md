# 🎮 Quiz Interativo para Programadores

Uma aplicação web interativa projetada para testar conhecimentos técnicos em desenvolvimento de software. O projeto destaca-se pela aplicação rigorosa de HTML5 semântico, técnicas modernas de estilização com CSS3 e foco na acessibilidade e usabilidade da interface.

## 💻 Sobre o Projeto

O objetivo deste projeto foi construir uma interface de questionário (quiz) completa e rica em detalhes, abordando diferentes formatos de perguntas e respostas. A identidade visual foi totalmente personalizada em tons terrosos (`#331F19`) e detalhes em verde (`#CCD595`), utilizando a fonte tipográfica *Cabin* para garantir uma leitura confortável.

### 🧠 Recursos e Conceitos Praticados:

* **HTML5 Semântico e Avançado:** Estruturação limpa utilizando tags que enriquecem a acessibilidade da página:
  * Tag `<abbr>` com `title` para fornecer o significado de acrônimos técnicos (HTML e CSS).
  * Estrutura `<figure>` e `<figcaption>` para a inclusão semântica de imagens com legenda.
  * Tag nativa interativa `<details>` e `<summary>` para criar um acordeão nativo que oculta/exibe o gabarito de respostas.
* **Ecossistema Completo de Formulários:** Implementação de 8 tipos diferentes de perguntas, utilizando uma vasta gama de controles nativos do navegador:
  * Seleção única (`radio`) e múltipla escolha (`checkbox`).
  * Entradas de texto livre (`text`), campos de segurança (`password`) e e-mail.
  * Seletores de data (`date`), menus suspensos (`select`/`option`) e upload de arquivos (`file`).
* **Estruturação de Dados com Tabelas:** Criação de uma matriz de pontuação semântica dividida corretamente entre cabeçalho (`<thead>`), corpo (`<tbody>`) e rodapé (`<tfoot>`), utilizando `colspan` para mesclagem inteligente de células.
* **CSS Moderno e Arquitetura de Layout:**
  * Uso de **Flexbox** (`flex-direction: column`) combinado com a propriedade `gap` para gerenciar os espaçamentos internos de forma automatizada e limpa.
  * Reset global com `box-sizing: border-box` para controle preciso de dimensões.
  * Customização de componentes nativos do sistema (como rádio e checkboxes) utilizando a propriedade moderna `accent-color`.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído de forma nativa com foco em performance e conformidade com os padrões web:

* **HTML5:** Estruturação semântica, acessibilidade e elementos de mídia.
* **CSS3:** Layout via Flexbox, importação de fontes externas (*Google Fonts*) e identidade visual personalizada.
