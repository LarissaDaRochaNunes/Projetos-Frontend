# 🎮 Jogo de Adivinhação Numérica

Uma aplicação web interativa desenvolvida para consolidar os fundamentos de lógica de programação e manipulação do DOM com JavaScript puro.

## 📝 Sobre o Projeto

O objetivo do jogo é adivinhar o número secreto gerado aleatoriamente pelo computador. A cada tentativa, a aplicação valida o palpite do usuário e fornece um feedback visual e textual em tempo real, informando se o número correto é **maior** ou **menor** do que o digitado.

### 🧠 O que foi praticado neste projeto:
* **Lógica de Programação:** Uso de estruturas condicionais (`if`, `else if`, `else`) para validação dos palpites.
* **Manipulação do DOM:** Captura de eventos de clique, leitura de inputs e atualização dinâmica de textos na tela sem recarregar a página.
* **Métodos Matemáticos:** Utilização das funções `Math.random()` e `Math.floor()` para a geração de números pseudoaleatórios dentro de um intervalo definido.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias web fundamentais:

* **HTML5:** Estruturação semântica da página.
* **CSS3:** Estilização e layout da interface (responsivo).
* **JavaScript (ES6):** Toda a lógica do jogo e interatividade.
