# 📝 Cadastro de DEVs

Uma aplicação web estática focada na estruturação e estilização minuciosa de um formulário completo para captação de dados de desenvolvedores. O projeto foi desenvolvido com foco no domínio de tags semânticas do HTML5 e na personalização visual de elementos de interface utilizando CSS puro.

## 💻 Sobre o Projeto

O objetivo deste projeto foi consolidar o conhecimento sobre a criação de formulários na web, abordando um cenário real de coleta de currículos ou perfis técnicos. A interface foi estilizada com uma identidade visual moderna em tons de roxo (`#59429d`), utilizando técnicas de profundidade e efeitos interativos para melhorar a experiência do usuário.

### 🧠 Recursos e Conceitos Praticados:

* **Diversidade de Inputs Web:** Implementação de uma ampla variedade de componentes nativos de formulários:
  * Entrada de textos padrão (`text`) e validação estrutural de correio eletrônico (`email`).
  * Seleção de opção única utilizando botões de rádio (`radio`).
  * Múltipla escolha de habilidades com caixas de seleção (`checkbox`).
  * Menu de listagem suspensa (`select` / `option`) com desativação temporária da opção padrão.
  * Campo para textos longos (`textarea`).
* **Validação Nativa:** Uso do atributo `required` do HTML5 para impedir o envio do formulário sem o preenchimento de dados essenciais (Nome, Sobrenome e E-mail).
* **Experiência do Usuário (UX) Estilizada:**
  * **Pseudo-classe `:focus`:** Feedback visual com alteração sutil na cor de fundo dos campos enquanto o usuário digita.
  * **Pseudo-classe `:hover`:** Efeito de transição de cor e inversão de relevo sombreado no botão "Concluído" ao passar o ponteiro do mouse.
  * **Profundidade Visual:** Aplicação direcionada de sombras (`box-shadow` e `text-shadow`) para conferir um acabamento refinado à interface.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais da web, sem o auxílio de frameworks externos:

* **HTML5:** Estruturação semântica e hierárquica dos elementos.
* **CSS3:** Estilização geral, posicionamento absoluto do botão de ação e tratamento de interações do usuário.
